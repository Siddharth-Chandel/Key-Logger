
# Keylogger

Keylogger is a tool which is used to record keystrokes by a user. This tool is for educational purpose only. Use it wisely.


## Requirements
- For Keylogger :
	- keyboard module : for analyzing the keystrokes
	- os module : for file management
	- time module : for getting required breaks
	- requests module : to communicate over internet
	- shutil : for downloading a image

- For Steganography:
	- Steghide (have to be in the path)

- Command for getting the logs.txt from the steged-image using steghide:
	- Do it in CMD:
		`steghide extract -sf /path/to/the/steged-image -p /passcode`
        
        here the passcode is : _`1234`_

## Adding steghide in a path
- Open properties of `This PC`.
- Click on `Advance System Settings`.

![Screenshot](https://github.com/Siddharth-Chandel/Key-Logger/blob/main/Images/properties.png?raw=true)

- Click on `Environment Variables`.

![Screenshot](https://github.com/Siddharth-Chandel/Key-Logger/blob/main/Images/environment%20.png?raw=true)

- Click on `Path` from above box.

![Screenshot](https://github.com/Siddharth-Chandel/Key-Logger/blob/main/Images/path.png?raw=true)

- Now, click on `Edit`.

![Screenshot](https://github.com/Siddharth-Chandel/Key-Logger/blob/main/Images/edit.png?raw=true)

- Click on `New`.

![Screenshot](https://github.com/Siddharth-Chandel/Key-Logger/blob/main/Images/new.png?raw=true)

- Add the path of steghide folder.

![Screenshot](https://github.com/Siddharth-Chandel/Key-Logger/blob/main/Images/path2.png?raw=true)

- Now, apply the changes.
- In some cases, it requires to restart the system to use the functionality.

## FAQ

#### Ques : It will work on other system if we make an ```exe``` file of it and send it to other ?
No, as it require a file called steghide to be in a path.

#### Ques : Can I use different image to do steganography ?
Yes, you can. You just need to put the image you want to use in a current working directory and rename it as ```image.jpg```.
